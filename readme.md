# Clay Sorter 3D

一款基于 Web 的本地多人互动 3D 形状分类游戏。具有可爱的“哑光黏土”风格，完全无需摄像头，支持电脑鼠标和手机触摸操作。

## 功能特点
- **视觉风格**: 深色护眼背景，马卡龙色系积木，哑光软胶材质。
- **操作方式**: 直观的拖拽操作 (Raycasting)。
- **游戏流程**: 
    1. 设置玩家人数。
    2. 轮流输入名字进行挑战。
    3. 60秒限时分类积分。
    4. 自动生成排行榜领奖台。

## 如何部署 (GitHub + Vercel)

1. **创建仓库**: 在 GitHub 上创建一个新仓库。
2. **上传文件**: 将 `index.html`, `style.css`, `script.js` 上传到仓库根目录。
3. **连接 Vercel**: 
    - 登录 [Vercel](https://vercel.com)。
    - 点击 "New Project"。
    - 导入刚才的 GitHub 仓库。
    - 点击 "Deploy"。
4. **完成**: 获得在线链接，发给朋友们即可在任何设备上打开玩耍。

## 本地运行
无需安装 Node.js 环境，直接使用 VS Code 的 **Live Server** 插件打开 `index.html` 即可运行。
